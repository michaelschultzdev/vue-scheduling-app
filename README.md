# vue-scheduling-app

To start, run:

`yarn && yarn dev`

Make sure to set up supabase and create a .env.local file with the following variables:

```
VITE_URL_SUPABASE=yoursupabaseurl.co
VITE_APIKEY_SUPABASE=your-supabase-anon-key
```
